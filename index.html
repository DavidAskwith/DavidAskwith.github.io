Custom Built Debian Linux


Previous Next

This is the primary platform that I use for developing applications. I built the system up from the Debian command line only install. The entire systems configuration files and install scripts are hosted on GitHub allowing me to either install the entire system anywhere or make use of individual configuration files for specific programs that I am using. The fact that I have had complete control of the design of the system from the ground has allowed me to not only keep the system extremely lightweight but also allows me to tailor the system to my specific style of computing. I have put a strong focus on building my environment with programs that are geared towards using keyboard shortcuts for all operations. This allows for less context switching and results in a far more efficient development work flow.

The application was quickly completely overhauled and hardly resembled the original created in the tutorial. This was a very valuable project for the progression of my web development skills.
